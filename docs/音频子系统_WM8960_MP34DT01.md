# 音频子系统技术规格书 — WM8960 + MP34DT01

## 一、概述

H753 主控板音频子系统由**输入**（MP34DT01 PDM 麦克风）和**输出**（WM8960 I2S 音频模块 + 喇叭）两部分组成。通过 DFSDM 和 SAI1 两个独立 DMA 通道并行工作，实现全双工音频链路。

```
[用户语音] → MP34DT01 → DFSDM(DMA) → USART3 → PC (实时流/KWS推理)
[语音播报] → SD卡WAV → SAI1(DMA) → WM8960 → 喇叭
```

## 二、硬件架构

### 2.1 引脚分配

| 外设 | 信号 | 引脚 | 方向 | 备注 |
|:---|:---|:---|:---|:---|
| **I2C1** | SCL | PB6 | 输出 | WM8960 寄存器配置, 100kHz |
| | SDA | PB7 | 双向 | 内部上拉已启用 |
| **SAI1** | FS_A (LRCLK) | PE4 | 输出 | 帧时钟, 48kHz |
| | SCK_A (BCLK) | PE5 | 输出 | 位时钟, ~1.536MHz |
| | SD_A (DACDAT) | PB2 | 输出 | 音频数据 → WM8960 **RXSDA** |
| | ~~MCLK_A~~ | PE2 | 未用 | WM8960 使用板载 24MHz 晶振 |
| **DFSDM1** | CKOUT | PC2 | 输出 | PDM 时钟, 2.46MHz |
| | DATIN0 | PC1 | 输入 | PDM 数据 |
| **USART3** | TX | PD8 | 输出 | 麦克风音频流, 3Mbps |

### 2.2 WM8960 模块关键连接

| WM8960 模块引脚 | STM32 引脚 | 说明 |
|:---|:---|:---|
| SCL (SCLK) | PB6 | I2C 时钟 |
| SDA (SDIN) | PB7 | I2C 数据 |
| CLK | PE5 | I2S 位时钟 |
| WS | PE4 | I2S 帧同步 |
| **RXSDA** | **PB2** | **音频数据输入 (关键! 非 TXSDA)** |
| VCC | 3.3V | 供电 |
| GND | GND | 共地 |

> **⚠️ 重要**: WM8960 模块的 DAC 数据输入脚是 **RXSDA**，而不是 TXSDA。TXSDA 是 ADC 数据输出脚（录音方向），接错会导致完全无声。

### 2.3 时钟方案

| 时钟 | 来源 | 频率 | 用途 |
|:---|:---|:---|:---|
| WM8960 MCLK | 模块板载 24MHz 晶振 | 24MHz | WM8960 SYSCLK, DAC 过采样 |
| SAI1 BCLK | STM32 PLL2P (86MHz) → MCKDIV/7 | ~1.536MHz | I2S 位时钟 |
| SAI1 LRCLK | BCLK / 32 | ~48kHz | I2S 帧时钟 (采样率) |
| DFSDM PDM_CLK | D2PCLK1 (120MHz) / 9 | 2.46MHz | 麦克风 PDM 时钟 |

WM8960 R4=0x0000 (MCLK/1, SYSCLK=24MHz), 满足 SYSCLK ≥ 256 × fs 的最低要求 (24MHz ≥ 12.288MHz)。

## 三、WM8960 驱动

### 3.1 软件架构

```
wm8960.h / wm8960.c
├── I2C 寄存器读写 (WM8960_Write_Reg / WM8960_Read_Reg)
├── 初始化 (WM8960_Init)
│   ├── 软件复位
│   ├── 上电序列 (VMID → DAC → 输出级)
│   ├── 音量/混音/Class D 配置
│   └── 时钟 + 音频接口配置 (I2S, 16-bit, Slave)
├── 播放控制 (WM8960_Play / WM8960_Stop)
│   ├── PLL2 自动启动
│   └── HAL_SAI_Transmit_DMA (Circular DMA)
├── 音量控制 (WM8960_SetVolume, 0-100)
└── SAI DMA 回调 (TxHalfCplt / TxCplt / ErrorCallback)
```

### 3.2 关键寄存器配置

| 寄存器 | 值 | 说明 |
|:---|:---|:---|
| R4 (CLOCK1) | `0x0000` | MCLK/1, SYSCLK=24MHz, Slave |
| R5 (ADC_DAC) | `0x0000` | DAC 不静音 |
| R7 (AUDIO_IF) | `0x0002` | I2S Philips, 16-bit, Slave |
| R25 (PWR1) | `0x01C0` | VMID 5kΩ startup + VREF |
| R26 (PWR2) | `0x01F8` | DACL/R + LOUT1/R + SPKL/R |
| R34/R37 (MIX) | `0x0180` | DAC → 左/右输出 |

### 3.3 SAI1 配置 (CubeMX)

| 参数 | 值 |
|:---|:---|
| Audio Mode | Master Transmit |
| Protocol | I2S Standard |
| Data Size | 16 Bits |
| Number of Slots | 2 |
| Output Drive | Enable |
| FIFO Threshold | Full |
| Clock Source | PLL2P (86MHz) |
| MCKDIV | 7 (86/7≈12.286MHz, fs≈48kHz) |
| DMA | Circular, Half Word ×2 |

### 3.4 API 接口

```c
int  WM8960_Init(void);                          // 初始化, 返 0 成功
int  WM8960_Play(uint16_t *buf, uint32_t size);  // 启动 Circular DMA
void WM8960_Stop(void);                           // 停止 DMA
void WM8960_SetVolume(uint8_t vol);               // 0-100 音量
int  WM8960_Write_Reg(uint8_t reg, uint16_t dat); // I2C 写寄存器
uint16_t WM8960_Read_Reg(uint8_t reg);            // 读寄存器缓存
```

### 3.5 调试记录

| 问题 | 根因 | 解决 |
|:---|:---|:---|
| 完全无声 | PB2 错接到 TXSDA (ADC 输出) | 改接 RXSDA (DAC 输入) |
| I2C 扫描失败 | 上电时序异常/R6 写入保留寄存器 | 优化上电顺序, 移除 R6 写入 |
| SAI Error 132/134 | BCLKDIV 不对/帧格式不匹配 | 回归 Slave 模式, FRL=31, SLOTSZ=16 |
| 旋律音高不对 | 51 FreqTable 与标准十二平均律映射错误 | 使用标准音高频率表 |
| 初始化后无声 | WM8960 SYSCLK 不满足 256×fs | MCLK 直通 (R4=0x0000), SYSCLK=24MHz |

## 四、MP34DT01 麦克风驱动

### 4.1 配置参数

| 参数 | 值 |
|:---|:---|
| 麦克风型号 | MP34DT01 (PDM MEMS) |
| 接口 | DFSDM1 Channel 0, Filter 0 |
| PDM 时钟 | 2.46MHz (D2PCLK1 / 9) |
| 滤波器阶数 | Sinc3 |
| 过采样率 | 64× |
| 输出数据率 | 38.4kHz (2.46M / 64) |
| 输出精度 | int32 (DFSDM 转换后 int16 重采样) |
| DMA | Circular, Word ×2, 双缓冲 |
| 数据流向 | DFSDM → DMA → 环形缓冲 → USART3 (3Mbps) → PC |

### 4.2 关键修复

- **DFSDM 时钟分频**: Divider=9 → PDM_CLK=2.46MHz (> MP34DT01 最低要求 1MHz)
- **数据转换**: int32 → int16 转换, 避免符号扩展噪声
- **实时流协议**: USART3 输出 `0xAA 0x55` 帧头 + PCM 数据, 兼容 `tools/mic_stream.py`

## 五、时钟树总览

```
HSE (8MHz)
  ├─ PLL1: ×120/2 → SYSCLK=480MHz
  ├─ PLL2: /8, ×172, /2 → PLL2P=86MHz → SAI1 kernel clock
  └─ D2PCLK1: 120MHz → DFSDM1 kernel clock

WM8960 板载 24MHz 晶振 → WM8960 MCLK
```

**DFSDM1 与 SAI1 使用不同的内核时钟源，互不干扰。**
