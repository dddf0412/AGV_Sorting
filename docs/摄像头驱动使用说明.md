# 摄像头驱动使用说明

文件: `camera.h` / `camera.c`
硬件: OV2640 (正点原子 ATK-MC2640), DCMI, SCCB 软件 I2C (PB6/PB7)

---

## 一、两种输出模式

| 模式 | JPEG (默认) | RGB565 |
|------|------------|--------|
| 用途 | PC 实时预览、调试 | 本地图像处理 (颜色/人脸识别) |
| 分辨率 | 320×240 | 320×240 (最大, 受限于内部 SRAM) |
| 帧率 | ~8fps (受 UART 波特率限制) | 30fps (DCMI 直存内存, 无传输瓶颈) |
| 数据位置 | `camera_tx_fb[]` → UART 发出 | `camera_fb[]` → 直接读像素 |
| 单帧大小 | 10-25KB (可变) | 150KB (固定) |

```c
/* 运行时切换 (无需重新初始化) */
Camera_SetFormat(CAMERA_FMT_JPEG);    // → PC 预览
Camera_SetFormat(CAMERA_FMT_RGB565);  // → 本地处理
```

## 二、数据流架构

```
OV2640 ──(DCMI 8-bit)──> camera_fb[] ──(中断回调)──> camera_tx_fb[]
                              │                            │
                              │ (RGB565 模式)              │ (JPEG 模式)
                              ▼                            ▼
                         直接读像素                     UART → PC
```

- **中断回调**: 每秒触发 ~30 次 (VSYNC)，快速复制帧到 `camera_tx_fb[]`，设 `camera_frame_ready = 1`
- **主循环**: 检查 `camera_frame_ready` → `Camera_DumpFrame()` 发送 → `Camera_RestartSnapshot()` 重启 DCMI

## 三、SCCB 通信

使用软件 I2C 位带操作 (PB6=SCL, PB7=SDA)，而非 HAL 硬件 I2C。

原因: OV2640 的 SCCB 协议不完全兼容 I2C (读操作时序差异)，硬件 I2C 初始化时连接不稳定。软件 I2C 仅用于启动时写 ~170 个寄存器，耗时约 200ms，不影响后续帧捕获性能。

## 五、API 接口

```c
int  Camera_Init(void);                          // 初始化 OV2640 + SCCB
int  Camera_Start(void);                         // 启动 DCMI SNAPSHOT 捕获
int  Camera_Stop(void);                          // 停止捕获
int  Camera_SetFormat(Camera_Format_t fmt);      // JPEG / RGB565
int  Camera_SetResolution(Camera_Resolution_t r); // QQVGA/QVGA/VGA/SVGA
int  Camera_DumpFrame(void);                     // 发送当前帧 (主循环调用)
void Camera_RestartSnapshot(void);               // 重启下一帧 (主循环调用)

extern volatile uint8_t camera_frame_ready;      // 1=新帧就绪
```

## 六、PC 实时预览

```bash
pip install pyserial opencv-python numpy
python tools/camera_viewer.py COMx [baud]
```

协议: `AA 55` + type[2] + size[4] + reserved[2] + JPEG_DATA[...]

JPEG 模式下 stm32 自动每帧发送，PC 端 OpenCV 窗口实时刷新。

## 七、后续切换到 RGB565 本地处理

```c
/* 在 App_Init 中 */
Camera_SetFormat(CAMERA_FMT_RGB565);

/* 在 App_Run 中 */
if (camera_frame_ready) {
    camera_frame_ready = 0;
    Camera_RestartSnapshot();

    // 直接处理 camera_fb[]
    // camera_fb 是 uint32_t[] 数组, 每像素 RGB565:
    //   uint16_t pixel = ((uint16_t*)camera_fb)[y * 320 + x];
    //   uint8_t r = (pixel >> 11) & 0x1F;
    //   uint8_t g = (pixel >> 5) & 0x3F;
    //   uint8_t b = pixel & 0x1F;
}
```
